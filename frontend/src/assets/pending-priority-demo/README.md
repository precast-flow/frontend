# Bekleyen iş / öncelik raporu — demo ürün görselleri

Bu klasöre **PNG, JPG, JPEG, WebP, AVIF, GIF veya SVG** dosyalarını ekleyin. Öncelik raporu ekranında **İkon** ve **Galeri** görünümleri bu dosyaları alfabetik dosya adına göre sırayla kullanır; satır sayısı görsel sayısından fazlaysa liste başa sarılır.

Geliştirme sunucusunu veya `npm run build` sonrası yeni dosyaların tanınması için projeyi yeniden başlatmanız gerekebilir (Vite `import.meta.glob` derleme anında toplar).

Örnek: `01-parca-a.png`, `02-parca-b.jpg`, …
